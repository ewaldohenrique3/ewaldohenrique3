# 👨‍💻 Ewaldo Henrique

**`Analista de Testes | Quality Assurance | QA`**

Me chamo Ewaldo Henrique Soares de Moura, tenho 25 anos e sou natural da Bahia. Atualmente moro em Contagem/MG e estou cursando Análise e Desenvolvimento de Sistemas pela UNINTER.

Sou apaixonado por tecnologia e estou construindo minha carreira na área de Qualidade de Software (QA), com foco em testes manuais e automação de testes.

Atualmente estou aprofundando meus conhecimentos em automação de testes utilizando Selenium WebDriver, Cypress e Robot Framework, trabalhando com as linguagens Java e JavaScript. Também estudo automação de testes de API com Rest Assured, banco de dados SQL e conceitos de CI/CD com GitLab e pipelines.

Meu objetivo é evoluir cada vez mais na área de QA, desenvolvendo soluções de testes automatizados que contribuam para a qualidade e confiabilidade de softwares.
